# CEP-78 NFT Bridge

Casper smart contracts for locking and releasing CEP78 NFTs for bridging between Casper and EVM chains like Etherem and BNB chain

## Getting started
### Requirements
- Rust
- make
### Build
```
make prepare
make build-contracts
```

## Test
```
make test
```
